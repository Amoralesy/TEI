<param>

<param> parameter of a model behaviour
Moduletagdocs — Documentation Elements
Attributesatt.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp))
name name¶Name of parameter Suggested values include: 1] alternate; 2] content; 3] default; 4] height; 5] id; 6] label; 7] level; 8] link; 9] place; 10] type; 11] url; 12] width
Status Required
Datatype teidata.xpath
Suggested values include:
alternate
expression evaluating to a content to process as alternative version (eg. for reg/orig pairs wrapped in choice); used with alternate behaviour
content
expression evaluating to a content to process by a behaviour (used with all behaviours with the exception of alternate and omit); when this parameter is not explicitly given on a model, currently processed element is used as default value
default
same as content but used only for alternate behaviour
height
expression that should evaluate to something that could be used as a value for height attribute on HTML img element to specify the height of the graphic, e.g. "300px", "50%"; used for graphic behaviour
id
expression that should evaluate to something that could be used as identifiers for an anchor; used for anchor behaviour
label
expression evaluating to something that could be used as label for breaks or footnote references (eg. concat('Page ', @n) would produce labels like 'Page 1', 'Page 2' etc; used with note or break behaviours
level
expression which should evaluate to a positive integer, used on heading to specify its structural level; for HTML outputs, this translates to h1, h2 etc.
link
expression pointing at the content to be used as URL for hyperlink generated with link behaviour
place
expression specifying placement of a note generated; if it evaluates to 'bottom', an anchor linking to a div containing the actual note is created, all others locations result in creation of a wrapper element holding the content of the note only ; used only on note behaviour; sample values include: "margin", "footnote", "endnote", "inline", "bottom"
type
specifies the kind of break to enact (when used on break behaviour) or index to generate (if used on index behaviour)
url
source of the graphic; in HTML outputs to be used as a value of src attribute on img element; used on graphic behaviour
width
expression that should evaluate to something that could be used as a value for width attribute on HTML img element to specify the width of the graphic, e.g. "300px", "50%"; used for graphic behaviour
value value¶supplies an Xpath expression used as the value for a parameter
Status Required
Datatype teidata.xpath
Contained by
tagdocs: model
May containCharacter data only
Note

The content of parameter is an XPath Expression. The names and datatypes of the expected parameters should be documented in the corresponding customization using a paramSpec element.

Example
<model predicate="sic and corr"
 behaviour="alternate">
 <param name="default" value="corr"/>
 <param name="alternate" value="sic"/>
</model>
<model predicate="abbr and expan"
 behaviour="alternate">
 <param name="default" value="expan[1]">
  <param name="alternate" value="abbr"/>abbr</param>
</model>
